ELEMENTOR TEMPLATE IMPORT
========================

Import these templates into Elementor for full design control:

1. Install Elementor (free): https://wordpress.org/plugins/elementor/
2. Go to Templates > Saved Templates > Import Templates
3. Upload the JSON files from this folder

Alternatively, create your own layouts:
- Edit any page with "Edit with Elementor"
- Use the theme's CSS classes (hero-banner, service-card, contact-cta, etc.) in your designs
- All design styles are in the theme - just add your content structure

For full demo design:
1. Run Import Demo from Appearance > Import Demo
2. Edit the Home page with Elementor
3. Add sections matching the design (Hero, About, Services, Testimonials, etc.)
4. Use Custom CSS or Elementor styling - colors are: --red #b71c1c, --brown-dark #3e2723
